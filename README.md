# Pain Quotidien - App ERPNext

Application ERPNext personnalisée pour l'association Pain Quotidien.

## Features

- Workspace Épicerie Solidaire
- Gestion bénéficiaires
- Gestion donateurs
- Suivi distributions

## Installation

```bash
bench get-app https://github.com/Pat-PQUOT/PAIN-QUOTIDIEN
bench --site votre-site install-app pain_quotidien
```
